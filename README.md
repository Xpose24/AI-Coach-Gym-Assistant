\# 🏋️ AI Coach Gym Assistant



An AI-powered fitness assistant that uses computer vision to analyze human body movements in real-time, track exercises, and provide automated feedback using pose estimation.



\---



\## 🚀 Features



\- 🎥 Real-time pose detection using webcam  

\- 📊 Automatic repetition counting  

\- 📐 Joint angle calculation for accuracy  

\- 🔥 Calorie estimation based on activity  

\- 💻 Interactive web interface using Streamlit  

\- 🌐 Public access using ngrok  



\---



\## 🧠 Technologies Used



\- Python  

\- OpenCV  

\- MediaPipe (Pose Estimation)  

\- Streamlit  

\- NumPy  

\- Ngrok  



\---



\## ⚙️ How It Works



1\. Captures live video from webcam  

2\. Uses MediaPipe to detect body landmarks  

3\. Calculates joint angles  

4\. Identifies exercise movements  

5\. Counts repetitions automatically  

6\. Displays results on web interface
